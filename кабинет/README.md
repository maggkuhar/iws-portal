# Блок: Личный кабинет

Blueprint: `cabinet`

## Маршруты
- `/register` — регистрация
- `/login` — вход
- `/logout` — выход
- `/cabinet` — личный кабинет
- `/cabinet/profile` — редактирование профиля
- `/cabinet/messages` — список диалогов
- `/cabinet/messages/<to_id>` — переписка

## Шаблоны
- `login.html`, `register.html`, `cabinet.html`
- `cabinet_profile.html`, `cabinet_messages.html`
