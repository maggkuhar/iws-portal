## Суть проекта

Портал бренда **IWS** (Interview with Subconscious — Интервью с подсознанием). Метод самопознания: онлайн (Telegram-боты, мобильное приложение) + офлайн (книги, дневники, МАК-карты, настольная игра, мерч). Портал = витрина бренда + магазин + личный кабинет + расписание офлайн-игр + сообщество + интеграция с TG-ботами.

## Архитектура / стек

- **Backend:** Flask + blueprints (как в `blogery`/`alexwake`)
- **БД:** PostgreSQL (предварительно — масштаб больше SQLite, см. описание)
- **Оплата:** ЮКасса / Тинькофф / Stripe (выбор по факту)
- **Хранилище медиа:** S3 или локальный сервер (решается по нагрузке)

- Реферальная и партнёрская программы

## Ключевые решения

- **Порядок разработки фиксирован:** Витрина → Магазин → Кабинет → Расписание → Сообщество → Интеграция с ботами → Мобильное приложение. Не перескакивать.
- **БД — PostgreSQL** (а не SQLite как в других проектах). Причина: масштаб (магазин с тысячами товаров, сообщество, сообщения) — SQLite не выдержит конкурентной записи.
- **Боты IWS уже существуют** (`@iwsgamebot`, `@iwsdiarybot`) — их не переписываем. Интеграция (блок 6) — синхронизация, не переезд.
- **Концепция зафиксирована 2026-04-06**, все продукты и блоки утверждены — не пересматривать без явной просьбы.
- **Архитектура — Flask blueprints**, тот же паттерн что в остальных проектах (один блок = одна папка-blueprint).
